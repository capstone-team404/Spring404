# Elevator Speech

## 프로젝트명 
  > 여성 1인 여행자를 위해, AI 기반 안전 분석으로 정보 공백을 해결하고 안전 경로를 추천하는 지도 서비스

---

## 문제 정의

### 고객
- 낯선 지역을 방문하거나 야간 이동 비중이 높은 여성 1인 여행자

### Pain Point
- 기존 지도 서비스는 **거리와 시간 중심**으로 경로 제공
- 해외 및 낯선 환경에서 **안전 정보 부족**
- 지역별 **체감 안전도 확인 어려움**
- 이동에 대한 **심리적 불안 및 제약 발생**

### Vitamin
- 데이터 기반 정보로 **신뢰도 높은 의사결정 가능**
- **안전한 경로 선택**으로 이동 부담 감소

---

## 해결 아이디어

### 1. 데이터 기반 안전도 산출 및 시각화
사용자 리뷰와 공공데이터를 결합하여 지역별 안전 점수를 산출하고,  
AI 분석을 통해 안전도를 정량화하여 지도 위에 직관적으로 시각화합니다.

### 2. 안전도 기반 경로 추천
산출된 안전 점수를 기반으로 위험 구간을 회피하고,  
보다 안전한 경로를 우선적으로 추천합니다.

---

## 기술/구현
| 단계 | 핵심 기술 | 수행 내용 |
| :--- | :--- | :--- |
| 데이터 수집 | Google Maps API | 전 세계 장소 데이터 및 사용자 리뷰(비정형 데이터) 추출 |
| AI 분석 | GPT-4o-mini | 리뷰 문맥 분석을 통해 CPTED(범죄예방환경설계) 기준의 기초 점수 산출 |
| 결함 보정 | 자체 가중치 로직 | AI의 판단 오류를 제어하기 위한 '위험 키워드 가중치 로직' 직접 구현 |
| DB 관리 | MySQL | AI 산출 점수와 사용자 평점 데이터를 구조화하여 저장 및 연동 |

## 우리 서비스의 차별성

#### 사용자 리뷰 + 공공데이터 기반 안전 점수
- 사용자 경험과 공공데이터를 결합해 안전 정보를 데이터화
- 분산된 후기를 구조화하여 활용 가능한 정보로 전환

#### AI 기반 지역 안전도 정량화
- 리뷰를 AI로 분석해 체감 안전 요소를 반영
- 지역별 안전 점수를 산출하고 실시간으로 갱신

#### 안전 중심 경로 추천
- 안전 점수를 기반으로 위험 구간을 회피하는 경로 제공
- 최단 경로가 아닌 ‘더 안전한 경로’ 제시
